## Overview

The `DoorNode` represents a highly configurable parametric door that can be placed on walls. It supports various door configurations including panel designs, glass segments, swing directions, and commercial hardware.

**Key features**:
- Wall-mounted placement with configurable side
- Segmented door leaf with panel, glass, or empty segments
- Configurable frame, threshold, and handle
- Swing direction and hinge side control
- Commercial hardware support (door closer, panic bar)

## DoorSegment Interface

<ParamField path="type" type="'panel' | 'glass' | 'empty'" required>
  Type of door segment.
  
  **Values**:
  - `"panel"` - Raised or recessed panel
  - `"glass"` - Glazed segment
  - `"empty"` - Flush flat fill
</ParamField>

<ParamField path="heightRatio" type="number" required>
  Proportion of the door height this segment occupies.
  
  **Example**: `0.4` (40% of door height)
  
  All segment heightRatios must sum to 1.0.
</ParamField>

<ParamField path="columnRatios" type="Array<number>" required>
  Horizontal division ratios for columns within the segment.
  
  **Default**: `[1]` (single column)
  
  **Example**: `[0.5, 0.5]` (two equal columns), `[1]` (no division)
</ParamField>

<ParamField path="dividerThickness" type="number" required>
  Thickness of dividers between columns in meters.
  
  **Default**: `0.03`
</ParamField>

<ParamField path="panelDepth" type="number" required>
  Depth of raised (+) or recessed (-) panel in meters.
  
  **Default**: `0.01`
  
  **Example**: `0.01` (raised 1cm), `-0.01` (recessed 1cm)
</ParamField>

<ParamField path="panelInset" type="number" required>
  Inset distance from segment edge to panel in meters.
  
  **Default**: `0.04`
</ParamField>

## Usage

### Creating a Standard Interior Door

```typescript
import { DoorNode } from '@pascal/core/schema/nodes/door'

const interiorDoor = DoorNode.parse({
  name: 'Bedroom Door',
  wallId: 'wall_bedroom',
  position: [2.5, 1.05, 0],
  width: 0.8,
  height: 2.0,
  hingesSide: 'left',
  swingDirection: 'inward',
})
```

### Creating a Glass Panel Door

```typescript
const glassDoor = DoorNode.parse({
  name: 'Office Glass Door',
  wallId: 'wall_office',
  width: 0.9,
  height: 2.1,
  segments: [
    {
      type: 'glass',
      heightRatio: 1.0, // Full height glass
      columnRatios: [1],
      dividerThickness: 0.03,
      panelDepth: 0.01,
      panelInset: 0.04,
    },
  ],
})
```

### Creating a Double Panel Door

```typescript

  wallId: 'wall_living',
  width: 0.9,
  height: 2.1,
  segments: [

    {
      type: 'panel',
      heightRatio: 0.5,
      columnRatios: [0.5, 0.5], // Two columns
      dividerThickness: 0.03,
```

## Door Configuration

Doors support extensive configuration:
- **Dimensions**: Width and height
- **Frame**: Thickness and depth
- **Threshold**: Optional with configurable height
- **Segments**: Multiple segments with different types
- **Hardware**: Handle, door closer, panic bar
- **Operation**: Hinge side and swing direction

## Coordinate System

Doors use wall-local coordinates:
- Position is relative to the wall's coordinate system
- Y coordinate represents vertical center of the door
- Side parameter determines which face of the wall
